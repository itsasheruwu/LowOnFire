# LowOnFire

Paper `1.21.11` plugin that lowers the on-screen fire overlay for Java clients and exports a matching Bedrock pack for Geyser/Floodgate setups.

## Build

```bash
./gradlew build
```

## Hosted Packs

```bash
./gradlew exportHostedPacks
```

That writes:

- `hosted/low-on-fire.zip`
- `hosted/low-on-fire-bedrock.mcpack`

The plugin's default Java pack URL points at the GitHub release asset for `v1.0.0`. If you change `appearance.height-scale`, rerun `./gradlew exportHostedPacks`, upload a new release asset, and update `java-pack.public-url` so the hosted file still matches the pack hash generated by the plugin.
